# 配置虚拟环境（官方 python， Conda 的方法不同，它有自己的一套逻辑，请查阅官方文档）
python -m venv env

# 激活虚拟环境
# linux/Mac os
source env/bin/activate
# windows(使用 cmd，或则在 powershell 使用 cmd 命令切换到 cmd 环境 )
cmd
call env\scripts\activate

# 安装依赖库

pip install -r requirements.txt

# 打包 exe

建议不要打包，不会提高性能，一般打包不会成功的，
而且，打包程序兼容并不高；如果您使用 conda的话，这个过程更加痛苦。

如果非要打包，建议安装较低一点版本的 PyQt5，比如 5.9.x 的版本，再次如果您使用conda，这个过程会更加痛苦。
